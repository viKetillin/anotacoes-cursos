# Estrutura de dados e Algoritmos com JS

## Por que é importante conhecer estrutura de dados? 
Conhecer estrutura de dados e algoritmos é importante, pois faz diferença no desempenho da aplicação, como também na qualidade do código.


## Dicas
- Em muitos projetos colocam o include do javascript na tag head, porém a melhor prática é adicionar no final da tag body, pois dessa forma, o navegador faz o parse do HTML, e ele será exibido antes de os scripts serem carregados. Com isso a página terá um melhor desempenho.
